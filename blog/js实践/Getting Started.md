# Getting Started

## 环境准备

### 基础环境

- node.js 6.6.0 https://nodejs.org/en/ 
- git https://git-scm.com

### 开发依赖

```shell
npm install -g gulp webpack grunt mocha
```

### 编辑器（任选）

- webstorm 
- sublime
- atom

### git clone 源码

- 获取项目git权限
- 生成公私钥
- 上传公钥
- 从http://192.168.0.2/#/dashboard/self拉取`front/composer`项目
- 切换到develop分支
