# Docker 学习笔记

## 概念介绍

### 某某即服务

#### IaaS

Infrastructure-as-a-Service（基础设施即服务）
 
厂商提供硬件设备的租赁
 
 
#### PaaS

Platform-as-a-Service（平台即服务）

也称为中间件，厂商提供虚拟服务器和操作系统
	
 
#### SaaS

Software-as-a-Service（软件即服务）

厂商提供应用服务

#### Container
